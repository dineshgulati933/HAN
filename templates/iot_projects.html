IoT & Arduino Projects

Smart agriculture and water monitoring solutions using IoT devices

Soil Moisture Monitoring System

Real-time soil moisture monitoring using Arduino and capacitive sensors. Data is transmitted via Wi-Fi to cloud platforms for analysis and visualization.

Features:
  • Real-time moisture readings
  • Automated irrigation triggers
  • Cloud data logging
  • Mobile app integration
Arduino ESP32 MQTT

Smart Weather Station

DIY weather station built with Arduino to monitor temperature, humidity, rainfall, and wind speed for agricultural decision-making.

Measurements:
  • Temperature & Humidity (DHT22)
  • Rainfall (tipping bucket)
  • Wind speed & direction
  • Solar radiation
Arduino Mega LoRa ThingSpeak

Automated Irrigation Controller

Smart irrigation system that automatically controls water delivery based on soil moisture levels and weather forecasts.

Capabilities:
  • Multi-zone control
  • Weather API integration
  • Schedule optimization
  • Water usage tracking
Arduino Relay Modules Blynk

Water Quality Monitoring

IoT-based water quality monitoring system for irrigation water, measuring pH, EC, TDS, and temperature.

Parameters:
  • pH levels
  • Electrical Conductivity (EC)
  • Total Dissolved Solids (TDS)
  • Water temperature
Arduino Uno pH Sensor ESP8266
