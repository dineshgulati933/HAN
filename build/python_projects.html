About These Applications

These Python web applications are deployed on cloud platforms (PythonAnywhere, Heroku) and provide interactive tools for:

  • Soil Water Balance Modeling: FAO-56 based simulations for irrigation management
  • Soil Organic Carbon Analysis: Assessment of carbon sequestration in agricultural systems
  • Field Data Management: Tools for synchronizing and managing field research data
